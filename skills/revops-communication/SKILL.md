# RevOps Communication Power

## Overview
Bridge the gap between "I built a flow" and "The SDR team knows how to use it". 

## Checklist
1. **Define Audience** — Leadership (ROI/Strategic), SDRs/AEs (How-to), Tech Team (SOP).
2. **Draft Content** — Use the `/revops-report` workflow structure.
3. **Visuals** — Generate Mermaid diagrams for every process flow.
4. **Deploy** — 
   - Confluence: `createConfluencePage`
   - Jira: `updateJiraIssue`
   - Slack: Format for copy-paste.

## Tone & Style
- **No Jargon:** Use "Automatic updates" instead of "Apex Trigger".
- **Benefit First:** "SDRs now save 2 hours/day" instead of "Implemented new API integration".
- **Quantifiable:** "1,200 records enriched with 94% accuracy".
